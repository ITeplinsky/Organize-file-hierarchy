# Organize-file-hierarchy

Написать скрипт, который парсит записи из https://generatedata.com/ (выбрать все поля, формат csv, на 2000 записей. 4 раза по 500 и удалить первую строку у всех файлов кроме первого, соединить их)
Скрипт должен получать на вход исходный csv файл и создавать корневую папку *result*, содержащую папку для каждой страны и субдиректорию для каждого региона

Пример: *Russia/Tatarstan*

В каждой папку региона должен содержаться ***${почтовыйкод}.yaml*** файл со следующим содержанием:

    code:
    person_1:
    name: ${имячеловека}
    address: ${адрес}
    curr: ${currency}

    person_2:
    name: ${имячеловека}
    address: ${адрес}
    curr: ${currency}
    
    person_3:
    name: ${имячеловека}
    address: ${адрес}
    curr: ${currency}

    
После создания структуры, скрипт должен упаковать полученную папку при помощи ***tar*** с форматом ***xz***
Если регионы не будут повторяться, то сгенерировать небольшой список регионов и вставить сперва в исходный файл вместо полученных

**Доп условие!** При создании папок, выводить сообщение зеленого цвета об успешности выполнения

Выполнить задачу используя инструменты:
***bash, cat, sed, awk, yq, envsubst, mkdir, printf, set*** и другие, удобные вам
